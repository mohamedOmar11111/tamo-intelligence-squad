# TAMO Protocol: /dialect (The Ammiya Engine)

**Objective:** Engineer data structures that bridge the "Linguistic Schism" between Modern Standard Arabic (MSA) and Egyptian colloquial intent to ensure maximum LLM vector proximity.

## 1. Squad Configuration
*   **The Linguist (Researcher):** Identifies the target sector's colloquial "Strike Terms" (slang, idioms, local pricing terms).
*   **The Vector Architect (Strategist):** Maps colloquial terms to their MSA counterparts and creates "Entity Bridges" for the Knowledge Graph.
*   **The Ghostwriter (Writer):** Produces "Hybrid Content" that is optimized for both search engine crawlers (MSA) and user conversion (Ammiya).
*   **The Auditor (Verifier):** Ensures the content maintains high-status authority while using local dialect.

## 2. Operational Logic
1.  **Phase Alpha (Extraction):** Scans "Dark Social" (WhatsApp/FB Communities) to extract raw Egyptian intent.
2.  **Phase Beta (Mapping):** Creates a technical glossary for the LLM context, defining local terms as primary entities.
3.  **Phase Gamma (Synthesis):** Re-writes product feeds or landing pages using the "Hybrid Protocol."
4.  **Phase Delta (Verification):** Tests vector proximity on multiple LLMs (ChatGPT, Gemini, Perplexity) to ensure brand citation.

## 3. Mission Standards
*   **Metric:** "Share of Model" - How often the AI uses local Cairene terms to describe your brand.
*   **Tone:** Authentic, High-Status, Culturally Precise.
